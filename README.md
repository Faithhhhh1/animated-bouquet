# 🌸 Animated Bouquet

A beautifully crafted animated floral bouquet built with **HTML5**, **CSS3**, **SVG**, and **Vanilla JavaScript**.

The project recreates a cinematic bouquet animation featuring organically growing stems, unfolding leaves, blooming flowers, floating particles, glowing fireflies, ambient lighting, and animated messages—all rendered entirely with **SVG**. No external frameworks or animation libraries are required.

---

## ✨ Features

- 🌿 Organic stem growth animation
- 🍃 Leaf unfolding animation

# 🎬 Animation Sequence

The bouquet animation follows this order:

1. Background fades in
2. Ambient lighting appears
3. Stems grow upward
4. Leaves unfold
5. Flowers bloom
6. Message fades in
7. Fireflies appear
8. Particles begin drifting
9. Idle breathing animation starts
